## Implementation of RS Proof

Below is the command to compile the C++ program:
```
g++ RS_Strategy.cpp -o RS_Strategy -std=c++14
```

Below is the command to run the C++ program:
```
RS_Strategy
```

Below are examples of the formula:
```
~(a>c)>(~(cvd)>(a^~c))
```
```
~(a>c)>(~(cvd)>(a^c))
```
```
(a>b)>((b>c)>(a>c))
```
```
(a>(avb))
```
